** Title: ** How to use Git Bash and the Node App 1) Accessing the Git Bash from vsCode: - open vsCode (https://code.visualstudio.com/download) - open a terminal session (across the top ribbon in vsCode, click terminal > new terminal) - Find the plus with a drop down arrow on the terminal (check top right of the terminal) - select Git Bash - you are now in the Git Bash inside of vsCode 2) Downloading node - From your browser launch, (https://nodejs.org/en/download/) - Download LTS version for your OS - Follow prompts - Close and re open vsCode if this was open when downloaded 3) Download express: - in the Git Bash, type this command npm -i express --save - this downloads express 3.5) Checking versions - To check versions in the git bash: --- for express, type: npm express -v --- for node, type: node --version --- for npm, type: npm --version 4) Changing working directory - To change working directory from Git Bash (preference: work from desktop) - for desktop, type: cd Desktop - to open a file in your desktop type: cd Desktop/filename - to go back, type: cd ../ - repeat above command as needed example: cd ../../../ 5) changing the display shown for working directory in Git Bash - To change the working directory, type: Ps1 = "$;" - Where $; (whatever is inserted inside of the quotes) will be the new name displayed 6) The Git Salute from the Git Bash: - Type these lines of code into the Git Bash git status git add filenames.ext seperatefileswithspaces.ext git commit filenames.ext seperatefileswithspaces.ext -m 'Message to push to GitHub with' git push
