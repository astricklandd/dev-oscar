** Title: ** How to use Git Bash and the Node App 1) Accessing the Git Bash from vsCode: - open vsCode (https://code.visualstudio.com/download) - open a terminal session (across the top ribbon in vsCode, click terminal > new terminal) - Find the plus with a drop down arrow on the terminal (check top right of the terminal) - select Git Bash - you are now in the Git Bash inside of vsCode 2) Downloading node - From your browser launch, (https://nodejs.org/en/download/) - Download LTS version for your OS - Follow prompts - Close and re open vsCode if this was open when downloaded 3) Download express: - in the Git Bash, type this command npm -i express --save - this downloads express 3.5) Checking versions - To check versions in the git bash: --- for express, type: npm express -v --- for node, type: node --version --- for npm, type: npm --version 4) Changing working directory - Clone repo from GitHub: click file > clone a repo > clone from GitHub > sign in when prompted > select repo - To change working directory from Git Bash (preference: work from desktop) - for desktop, type: cd Desktop - to open a file in your desktop type: cd Desktop/filename - to go back, type: cd ../ - repeat above command as needed example: cd ../../../ 5) changing the display shown for working directory in Git Bash - To change the working directory, type: Ps1 = "$;" - Where $; (whatever is inserted inside of the quotes) will be the new name displayed 6) The Git Salute from the Git Bash: - Type these lines of code into the Git Bash git status git add filenames.ext seperatefileswithspaces.ext git commit filenames.ext seperatefileswithspaces.ext -m 'Message to push to GitHub with' git push 7) Additional Information - useful Git Bash commands: https://education.github.com/git-cheat-sheet-education.pdf https://www.hostinger.com/tutorials/basic-git-commands - vsCode tips: https://www.desuvit.com/11-vscode-keyboard-shortcuts-that-will-boost-your-productivity/ https://code.visualstudio.com/docs/getstarted/tips-and-tricks - Shortcuts on keyboard: https://www.popsci.com/windows-keyboard-shortcuts/ https://1ttv8k49lumh3j429gbyykm1-wpengine.netdna-ssl.com/wp-content/uploads/2017/08/shoflo-learn-keyboardshortcutmodal1.png
